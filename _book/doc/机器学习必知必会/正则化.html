正则化

正则化

综述

机器学习中经常会在损失函数中加入正则项，称之为正则化Regularize。

目的：防止模型过拟合

原理：在损失函数上加上某些规则（限制），缩小解空间，从而减少求出过拟合解的可能性

例子：以最简单的线性模型为例

y=Xβ+ϵ y=X\beta + \epsilon y=Xβ+ϵ 我们在统计学习中接触到的最小二乘估计利用均方误差最小的原则来进行参数估计：

β^=argminβ∣∣y−Xβ∣∣2 \hat{\beta}=arg \min_{\beta}||y-X\beta||^2 ​β​^​​=arg​β​min​​∣∣y−Xβ∣∣​2​​ 在最小二乘估计中加入L2L2L2正则项后，我们得到岭估计：

β~=argminβ∣∣y−Xβ∣∣2+λ∣∣β∣∣2 \tilde{\beta}=arg\min_{\beta}||y-X\beta||^2+\lambda||\beta||^2 ​β​~​​=arg​β​min​​∣∣y−Xβ∣∣​2​​+λ∣∣β∣∣​2​​ 在数学上我们可以证明岭估计的参数模要严格小于最小二乘估计的参数模，换句话说，我们可以认为加入L2L2L2正则项后，估计参数长度变短了，这在数学上被称为特征缩减shrinkage。

shrinkage方法介绍：指训练求解参数过程中考虑到系数的大小，通过设置惩罚系数，使得影响较小的特征的系数衰减到000，只保留重要特征的从而减少模型复杂度进而达到规避过拟合的目的。常用的shinkage的方法有Lasso（L1L1L1正则化）和岭回归（L2L2L2正则化）等。 采用shrinkage方法的主要目的包括两个：一方面因为模型可能考虑到很多没必要的特征，这些特征对于模型来说就是噪声，shrinkage可以通过消除噪声从而减少模型复杂度；另一方面模型特征存在多重共线性（变量之间相互关联）的话可能导致模型多解，而多解模型的一个解往往不能反映模型的真实情况，shrinkage可以消除关联的特征提高模型稳定性。

总结

梳理一下，正则化有多种方式，包括L0 L0 L0（向量中非零元素个数），L1 L1 L1（向量中元素绝对值之和），L2 L2 L2（向量的模）。但是L0 L0 L0范数的求解是个NP完全问题，而L1 L1 L1也能实现稀疏并且比L0 L0 L0有更好的优化求解特性而被广泛应用。

L2 L2 L2范数指各元素平方和后开根的值，可令w w w每个元素接近于0，虽然不如L1 L1 L1更彻底地降低模型复杂度，但是由于处处可微降低了计算难度。
